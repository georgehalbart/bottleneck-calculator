🚀 PC Bottleneck Calculator

A lightweight browser extension that helps you analyze performance imbalance between CPU and GPU components. Quickly identify bottlenecks and make smarter upgrade decisions.

📌 Overview

PC Bottleneck Calculator is designed for developers, gamers, and PC enthusiasts who want to understand how well their hardware components work together. It provides fast and simple bottleneck analysis directly in the browser—no heavy software required.

⚙️ Features
🔍 Automatic bottleneck detection (CPU vs GPU)
🛠 Manual hardware comparison mode
⚡ Fast, browser-based calculations
🎯 Clean and user-friendly interface
💻 Supports modern CPUs and GPUs
🎯 Why Use This?
Identify performance limitations in your PC
Improve gaming performance and FPS stability
Make better hardware upgrade decisions
Avoid unnecessary spending on mismatched components
🧩 Installation
Chrome
Download or clone this repository
Go to chrome://extensions/
Enable Developer Mode
Click Load unpacked
Select the extension folder
Firefox
Go to about:debugging
Click This Firefox
Click Load Temporary Add-on
Select the manifest.json file
🚀 Usage
Open the extension
Select your CPU and GPU
Choose:
Automatic analysis
Manual comparison
View bottleneck results instantly
🛡 Privacy

This extension:

❌ Does NOT collect user data
❌ Does NOT track browsing activity
❌ Does NOT send data to external servers

All calculations are performed locally in your browser.
